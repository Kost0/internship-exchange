# Биржа стажировок

Пространство, которое объединит компании, нуждающиеся в молодых специалистах, и студентов полных амбиций

## Функционал

Компания может:
- Выкладывать вакансии стажировок
- Получать отклики на них
- Просматривать профили или прикрепленные резюме студентов

Студент может:
- Найти стажировку, которая подходит именно ему
- Рассказать о своих навыках
- Увидеть какие навыки от него требует бизнес

## Стек

- Бэкенд: Go
- Базы данных: PostgrSQL, MinIO
- Очередь: RabbitMQ
- Фронтенд: React + Vanilla JS
- Деплой: Docker Compose
